Phases 1 & 2
What's Covered — Define & Prove
Discovery, then prototype & validate
Phase1

Discovery & Strategy

Lock the targets, data sources, success metrics, and the data-quality bar before any build.

  • Strategy brief + confirmed persona definitions across the three categories + channel partners
  • Recommended data sources, tooling, and channel mix
  • Success metrics + the explicit data-quality criteria Phase 2 must hit
Phase2

Prototype in Claude Code — validate the data

Build a working prototype of the pipeline directly in Claude Code — pull, enrich, score, and draft against real records — with Ravi engaging directly when judgment or review is needed. Then review the results with VWC to confirm we are getting the right investors and the right data before investing in production.

  • Working Claude Code prototype of the identify → enrich → score → draft flow
  • Sample enriched + scored lead sets, by category, for VWC review
  • Data-quality findings + tuning of sources, filters, and scoring
  • Go / no-go checkpoint with VWC before Phases 3–4

Phase 2 is the de-risking step: it proves the data and scoring are right on real records before a dollar goes into the production server and web app. VWC sees actual prospects, not slideware.

Phase 3 — Production
What's Covered — The Production Engine
Phase 3 — the AI automation server
Once VWC signs off on the prototype data, we productionize it as a standing, hosted AI automation server — running continuously with human-in-the-loop approval, with AI delivered through Technijian Chat.AI.

1 · Agent + Agent Harness

Reasoning core + runtime

The validated logic deployed as an Agent on the Agent Harness — hosted, scheduled, stateful, observable — so it runs without a human babysitting it.

2 · Skills

Reusable capabilities

Identify, enrich, score, and draft packaged as discrete Skills — versioned and testable, each improvable independently.

3 · N8N workflow automation

Orchestration + approvals

Runs the end-to-end pipeline on a schedule and routes records through the human-in-the-loop approval gate before anything sends.

4 · Technijian Chat.AI

AI connectivity + tokens

All model calls route through Technijian Chat.AI for AI connectivity and metered token usage — one governed, observable AI layer.

5 · CRM + deliverability

HubSpot or SuiteCRM

Connect VWC's existing HubSpot, or have Technijian host, customize & brand SuiteCRM (crm.visionwisecapital.com); scored leads land with source/score/rationale and disposition flows back. Dedicated domain (SPF/DKIM/DMARC), warm-up, suppression, CAN-SPAM opt-out.

6 · Monitoring

Health + KPIs

Server health monitoring/alerting plus the KPI data that powers the web-app dashboard (Phase 4).

Hosted and operated by Technijian on an ongoing basis (see Commercials), documented for full transparency, and built to hand off.

Guardrails & Next Steps
Guardrails
Built compliant by design

How the engine respects VWC's regulatory reality

  • VWC is a private real estate sponsor — not a registered investment adviser or broker-dealer. The system never performs licensed activity.
  • Offerings are Reg D Rule 506(c): general solicitation is permitted, so outreach is allowed — but every investor's accredited status must be verified before any offering action.
  • Accredited investors only — targeting and gating are designed around that constraint end-to-end.
  • Human-in-the-loop — VWC reviews and approves every message in the web app before it sends; nothing goes out at scale unattended.
  • Anti-fraud discipline (no misleading or unsubstantiated performance claims), CAN-SPAM opt-out, TCPA, and state blue-sky notices respected.
  • Investor NPI is governed by NDA-VWC-001. VWC owns legal/compliance and engages securities counsel; Technijian flags considerations but does not give legal advice.

To start the build

  1. Approve this scope & sign the fixed-fee build SOW (build fee + monthly hosting; supersedes the prior advisory SOW).
  2. Confirm the empowered point of contact for scope, spend, and outreach approval.
  3. Line up usage budget (Technijian Chat.AI tokens, data, CRM) — finalized with you in Phase 1.
  4. Name securities counsel for outreach-content review and the 506(c) verification approach.
